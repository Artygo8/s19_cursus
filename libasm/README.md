# libasm

Small asm library made for nasm macho64. These functions are supposed to work like their relatives from the libc.

- [x] ft_read.s
- [x] ft_write.s
- [x] ft_strlen.s
- [x] ft_strcpy.s
- [x] ft_strcmp.s
- [x] ft_strdup.s
- [x] ft_putstr.s
